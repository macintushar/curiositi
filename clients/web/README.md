# Curiositi Client - Web

This is the web client for Curiositi. It is built with Next.js and Tailwind CSS, initially scaffolded with (Create T3 App)[https://create.t3.gg/].

## Development

```bash
cd clients/web
bun install
bun run dev
```

## Deployment

```bash
cd clients/web
bun run build
bun run start
```
